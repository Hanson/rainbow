# Rainbow

A skin for [laravel-admin](https://laravel-admin.org/)

## Install

`composer require hanson/rainbow:dev-master -vvv`

## Skins

rainbow-blue rainbow-green

publish resources

`php artisan vendor:publish --tag=rainbow-blue --force` 

## Configuration

```php
// admin.php
<?php

return [
   // ...
  'skin' => 'rainbow-blue',  
];
```
